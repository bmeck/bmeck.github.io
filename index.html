Self extracting Node.js binaries

These binaries add support for using source code bundling. These binaries will check for trailing zip archives in themselves and extract to temporary directories to run.

how to

   # create our zip
   # will be opened as if you had called `node $PATH_TO_MY_MODULE`
   zip -r bundle.zip $PATH_TO_MY_MODULE

   # add trailing zip archive to extracting node
   cat node_extracting bundle.zip > node_bundle

   # make bundled node executable
   chmod +x node_bundle

   # run
   ./node_bundle
